1、安装需要nodejs

2、进入Node.js目录下，执行命令npm install -g grunt-cli

3、git clone git://github.com/mobz/elasticsearch-head.git

4、cd elasticsearch-head/

5、进入head主目录执行npm install grunt

6、grunt -version
grunt-cli v1.3.2
grunt v1.0.1

7、npm install

8、npm run start 或者 grunt server

9、http://localhost:9100

10、cd elasticsearch-6.8.3/

11、vim config/elasticsearch.yml
http.cors.enabled: true 
http.cors.allow-origin: "*"
